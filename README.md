# vibe_coding_chiz_pyzh

Веб-приложение на TypeScript, которое проигрывает «Чижик-Пыжик» с транспозицией в полутонах и показывает визуализацию из 12 нотных секторов.

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Готовые статические файлы будут в `dist/`.

## Деплой на GitHub Pages

1. Выполнить `npm run build`.
2. Опубликовать содержимое `dist/` в ветку Pages (`gh-pages` или `docs/`), в зависимости от выбранной схемы в репозитории.
