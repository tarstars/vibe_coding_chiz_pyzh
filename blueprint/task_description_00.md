# Приложение
* type script, html, css
* разворачивается на github pages

# Что делает
* даёт возможность пользователю проиграть транспонированную мелодию Чижик-Пыжик

# Описание интерфейса
* есть поле ввода с кнопками "вниз-вверх" рядом
* поле ввода позволяет указать целое (воможно отрицательное) число
* кнопка, запускающая проигрывание чижика-пыжика

# Пользовательский опыт
* пользователь видит:
  * неактивный элемент, визуализирующий мелодию
  * кнопку проигрывания
  * поле ввода
* пользователь меняет число в поле ввода, при этом во внутреннем кольце элемента, визуализирующего мелодию, двигается стрелка, показывая где, после транспозиции, будет находиться нота "До"
* мелодия одна, фиксированная
* мелодия лежит в файле cp.abc, во время написания программы её можно перевести в вид [(полутон, длительность), ...]

# Описание элемента, визуализирующего мелодию

## Вид элемента
* эскиз элемента - в файле music_visualize.png
* элемент состоит из внутреннего круга и 12 секторов
* каждый из 12 секторов помечен нотой в нотации "латинская буква + знак альтерации"
* во внутреннем круге находится стрелка, показывающая, где будет находиться нота "до" после транспозиции на количество полутонов, введённых пользователем
* семь нот без знака альтерации нарисованы чистыми цветами радуги в стандартном порядке: до -> си = красный -> фиолетовый
* нота, расположенная между двумя основными, рисуется смешанным цветом её соседей по аддитивному правилу

## Как работает проигрывание мелодии
* нота "чижика-пыжика" сдвигается на указанное пользователем количество полутонов и проигрывается с правильной длительностью
* когда звучит нота, подсвечивается соответствующий сектор элемента, визуализирующего мелодию
* во время звучания ноты на внешней стороне появляется сектор, соответствующий ноте. Толщина сектора пропорциональна длительности ноты
